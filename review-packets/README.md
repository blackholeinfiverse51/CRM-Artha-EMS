# Review Packets

This folder contains structured review packets for planning, technical validation, QA/UAT, security, and release sign-off.

## Packet List

1. [01_EXECUTIVE_REVIEW_PACKET.md](./01_EXECUTIVE_REVIEW_PACKET.md)
2. [02_TECHNICAL_REVIEW_PACKET.md](./02_TECHNICAL_REVIEW_PACKET.md)
3. [03_QA_UAT_REVIEW_PACKET.md](./03_QA_UAT_REVIEW_PACKET.md)
4. [04_SECURITY_OPERATIONS_REVIEW_PACKET.md](./04_SECURITY_OPERATIONS_REVIEW_PACKET.md)
5. [05_RELEASE_SIGNOFF_PACKET.md](./05_RELEASE_SIGNOFF_PACKET.md)

## Suggested Usage Order

1. Executive alignment and business scope
2. Technical architecture and code readiness
3. QA and UAT validation
4. Security and operational controls
5. Final sign-off and deployment go/no-go

## Components Covered

- Core application: client and server
- CRM module: ai-crm
- Finance module: artha-finance
- Agent module: employee-agent

## Quick Start

1. Open packet 1 and set review date, owner, and scope.
2. Work through packet 2 and packet 3 with engineering and QA.
3. Complete packet 4 with security and operations.
4. Finalize packet 5 for release sign-off.
