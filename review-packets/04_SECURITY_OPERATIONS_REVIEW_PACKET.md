## Security Checklist

- [ ] Authentication and authorization paths verified.
- [ ] Role checks exist on sensitive endpoints.
- [ ] Input validation and sanitization reviewed.
- [ ] Secrets are not hardcoded in source files.
- [ ] Sensitive logs/data exposure risks reviewed.
- [ ] File upload restrictions and scanning strategy reviewed.

## Privacy and Compliance

- [ ] Data retention expectations are documented.
- [ ] Audit logging for critical actions is available.
- [ ] Employee monitoring/privacy disclosures reviewed.
- [ ] Access to personal data follows least privilege.

## Operations Checklist

- [ ] Environment variables documented and available.
- [ ] Backup/restore procedures validated.
- [ ] Monitoring and alerting are configured.
- [ ] Error dashboards/log pipelines are accessible.
- [ ] Incident response ownership is clear.

## Reliability and Recovery

- [ ] Health checks verified for critical services.
- [ ] Restart behavior validated.
- [ ] Rollback procedure tested in non-production.
- [ ] Dependency outages have fallback handling.
