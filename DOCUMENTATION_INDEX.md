## 🛠️ Automation Scripts

### [START_INTEGRATED_SYSTEM.bat](./START_INTEGRATED_SYSTEM.bat)
**Purpose:** One-click system startup  
**Usage:** Double-click to run  
**What it does:**
- Starts Workflow Backend (Port 5001)
- Starts AI CRM Backend (Port 8000)
- Starts AI CRM Frontend (Port 3000)
- Opens browser automatically

### [VERIFY_INTEGRATION.bat](./VERIFY_INTEGRATION.bat)
**Purpose:** Integration verification  
**Usage:** Double-click to run  
**What it checks:**
- Node.js installation
- MongoDB configuration
- Environment files
- Integration routes
- Frontend modifications
